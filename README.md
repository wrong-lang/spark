# Usage
You must hovering the text you want to convert before executing any functions

- `CTRL+SHIFT+F5` - Convert Thai <-> Eng

# Supported Layout

### Thai Layout
- Kedmanee
- Pattachote
- Manoonchai


### English Layout
- Qwerty
- Dvorak
- Colemak
